How to use this ding dang thing


key naming convention for global values:

*TYPE*_*SIGNATURE*_*NAME*

For example, if I'm making a trait for a civilization:

TRAIT_CIV * _EBW_ * CUSTOM_TRAIT_ONE

For 'local' values, that only exist in a specific file

-- *SIGNATURE*_*MOD_TARGET*_*INSTANCE_NUM*
-- EBW _CIV_TRAIT_CUSTOM_ONE _MOD_TWO
